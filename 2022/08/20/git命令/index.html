Git

Git 是一个分布式版本控制系统

版本控制系统 : version control system

  • 本地VCS
  • 集中式VCS ===> SVN 淘汰
  • 分布式VCS ===> Git 主流

**作用： ** 代码存档备份， 支持多人协作。

下载： https://git-scm.com/download/win

终端： 命令行窗口 terminal ==> 右键git bash here

清屏 ： clear / cls

  3. git status 查看状态 ==> 红色工作区 绿色 暂存区

  5. git commit -m “” 提交版本更新， 只要提交后，这个状态永久保存了， 不会丢失。
    1. 每写一个小功能， 就提交一次。

  7. git reset –hard 版本ID 回退到某个版本
